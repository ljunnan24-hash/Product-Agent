# Product Agent Examples

这些样例用于本地 Beta 的首次试用。运行 `pnpm local` 打开首页后，点击对话里的“载入示例”，系统会把样例材料填进上传区，不需要你先准备 README、PDF 或 CSV。

## local-beta-demo

`examples/local-beta-demo` 是一个虚构产品 SignalShelf 的分析包，包含：

- `README.md`：产品定位、目标用户、功能、早期牵引和风险。
- `interview-notes.txt`：12 个早期用户访谈的整理。
- `waitlist-metrics.csv`：早期渠道和激活数据。

建议验证问题：

> 请判断 SignalShelf 是否适合作为 local-first 产品分析工具继续推进。重点看：目标用户是否足够锐利，weekly decision memo 是否是好 wedge，下一步应该验证什么。

这个样例刻意包含支持证据和反对意见，便于检查 Product Agent 是否会区分材料证据、外部网页证据、缺失证据和下一步实验。
